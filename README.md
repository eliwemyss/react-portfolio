## Local development & visual updates

To run locally:

```bash

```

Summary of visual/accessibility updates applied:
- Added Google Fonts and improved typographic scale
- Polished header with semi-transparent backdrop and improved nav underline
- Hero banner: glass panel + separator animation + CTAs (See Projects, Get in Touch)
- Projects: responsive grid, reveal animation, hover lift, improved buttons
- Accessibility: focus-visible outlines for keyboard users, clearer focus for links/buttons

If you run into errors like `react-scripts: command not found`, remove `node_modules` and `package-lock.json` then run `npm install` again.
